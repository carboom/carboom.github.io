MySql

Posted on 2018-04-21

这是MYSQL

数据库本身是一个文件系统，格式化数据存储。
数据库管理系统：用户访问数据库中表内数据的中间部分。
数据库中以表为组织单位存储数据。字段对应数据类型。
java与关系型数据的对比：类–表 类的值–字段 对象–记录行。

常见数据库

  • MySql：开源免费，Oracle以收购，6.x版本开始收费。
  • Oracle：收费大型数据库。
  • DB2：银行系统多用。
  • SQLserver：Microsoft公司数据库，c#，.net使用。
  • SQLite：手机端的小型数据库。

SQL语句

结构化查询语言（Structured query language），关系型数据库管理系统都遵循的规范。

分类

  • 数据定义语言： DDL（data definition language）：create ，alter ，drop等。
  • 数据操作语言： DML（data manipulation language）：insert，delete，uptate等。
  • 数据控制语言：DCL（data control language）： 定义数据库的权限，安全级别。
  • 数据查询语言：DQL（data query language）：select，from，where等。

类的值–字段

  • int（integer） 整型 ；tinyInt 很小的整数； smallint 小的整数； mediumint 中等大小的整数
  • double 浮点型； float 单精度的浮点数 ；decimal（m，d）压缩严格的定点数
  • 文本、二进制类型 char(0/255) ;varchar(0/65535);tinyblob允许0/255字节；blob允许0/65535字节；mediumblob允许0/167772150字节；longblob允许0/4294967295字节；tinytext允许0/255字节；text允许0/65535字节；mediumtext允许0/167772150字节；longtext允许0/4294967259字节；varbinary（m）允许0到m个字节变长字节字符串；binary（m）允许0到m个字节定长字节字符串；
  • 日期型 date 格式 ：yyyy-mm-dd ；year 格式：yyyy；time 格式：hh–mm–ss；datetime 格式：yyyy–mm–dd hh–mm–ss；timestamp 格式：yyyy–mm–dd HH–mm–ss

DDL

create database db character set encoding；
show databases/create database db/tables ；
drop database db；drop table tb；
use db；
select database（）；
desc tb；
alter table tb add/modify/change/drop 列 类型 约束

DML

insert into tb values();
update tb set x=x… where condition;
delete from tb where condition;truncate table tb;

约束

primary key 主键约束，包含唯一值，不能包含null，每个表只能有一个主键。constraint命名主键别名。
auto_increment自动增长列，必须是整型，一般主键上加。
not null 强制列不接受null值。
unique 唯一标识
foreign key 外键约束
